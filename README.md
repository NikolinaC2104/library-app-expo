Library App (React Native + Expo)
-> jednostavna mobilna aplikacija za upravljanje knjigama u knjižnici, izrađena s React Native-om (Expo) i spojena na MockAPI backend.

Funkcionalnosti
-> Prikaz popisa knjiga
-> Dodavanje novih knjiga
-> Uređivanje postojećih knjiga
-> Brisanje knjiga
-> Praćenje dostupnosti knjiga

Tehnologije
-> React Native (Expo)
-> React Navigation
-> MockAPI
-> TypeScript
